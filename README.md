## Описание
Проект состоит из двух лабораторных работ:

Lab1 - Обработка изображений
Реализует интерфейс для обработки изображений с помощью библиотеки OpenCV.
Доступные методы: свёртка, преобразование в grayscale, гамма-коррекция, обнаружение границ, углов и окружностей.

Lab2 - Работа с API изображений животных
Загружает изображения животных через API, обрабатывает их (обнаружение контуров) и сохраняет результаты. Реализует операции сложения и вычитания изображений.

## Использование
Lab1 - Обработка изображений
Запуск программы осуществляется через командную строку:

python main_lab1.py <метод> <путь_к_изображению> [-o путь_для_сохранения] [--gamma значение]
Доступные методы
   -convolution — свёртка изображения с ядром
   -grayscale — преобразование в оттенки серого
   -gamma — гамма-коррекция (требует параметр --gamma)
   -edges — обнаружение границ (оператор Собеля)
   -corners — обнаружение углов (алгоритм Харриса)
   -circles — обнаружение окружностей (преобразование Хафа)
## Пример Lab1
Свёртка изображения:
python main_lab1.py convolution input.jpg

Lab2 - Работа с API изображений животных
Запуск программы:

python main_lab2.py [--limit N] [--output_dir путь]

Параметры Lab2
--limit - количество изображений для загрузки (по умолчанию: 3)
--output_dir - директория для сохранения результатов (по умолчанию: processed_images)

## Примеры Lab2
Загрузка 5 изображений:

python main_lab2.py --limit 5

## Требования
